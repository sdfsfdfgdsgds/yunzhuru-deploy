说明：验证卡密合法性，支持异步回调处理结果。

  • url：验证接口地址
  • card：用户输入的卡密
  • callbackFunc：回调函数名称，需接收2个参数参数为 ok: boolean 和 msg: string

function onCardVerify(success, message) {
          if (success) {

          }
        }

说明：验证卡密合法性，支持异步回调处理结果。

示例：window.Android.verifyCardKey2('{#id#}', 'https://yunzhuru.com/kami/', '你的卡密', 'onCardVerify')

参数：

  • {#id#}：当前弹窗 ID，会被系统自动替换，直接传这个变量即可
  • url：验证接口地址
  • card：用户输入的卡密
  • callbackFunc：回调函数名称，需接收4个参数，参数为 success: 状态 / statusCode: 状态码 / body: 响应正文 / headers: 响应头

回调示例：

function onCardVerify(success, statusCode, body, headers) {
      if (success && body === 'ok') {
        window.Android.writeSP("kami", cardInput.value);
        showResult('验证成功');
        setTimeout(() => window.Android.close('{#id#}'), 1000);
    
        // 如果响应头中包含卡密剩余信息，可以显示它
        if (headers.includes("卡密剩余")) {
          const match = headers.match(/卡密剩余\d+天/);
          if (match) {
            showResult(match[0]); // 弹出“卡密剩余X天”
          }
        }
    
      } else {
        showResult(body || '验证失败');
      }
    }

说明：通过原生发送网络请求（GET/POST），并将响应回调给 JS。

示例：window.Android.http('{#id#}', 'https://api.xxx.com/data', 'GET', '{}', '', 'onHttpResponse')

参数：

  • {#id#}：当前弹窗 ID，会被系统自动替换，直接传这个变量即可
  • url：请求地址
  • method：请求方式（"GET" 或 "POST"）
  • headers：请求头（JSON 字符串，如 { "Authorization": "Bearer token" }）
  • body：请求体（仅 POST 支持）
  • callbackFunc：回调函数名称，参数为响应信息

回调格式：响应内容将以 JSON 字符串形式传入，如：

{
          ok: true,
          status: 200,
          headers: { "Content-Type": "application/json" },
          body: "响应文本"
        }

回调示例：

function onHttpResponse(result) {
          const res = JSON.parse(result);
          if (res.ok) {
            console.log('响应成功：', res.body);
          } else {
            console.error('请求失败：', res.body);
          }
        }
